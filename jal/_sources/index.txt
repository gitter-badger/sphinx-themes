Welcome to Jal's documentation!
===============================

About
-----
Jal is a theme for the Python_ Sphinx_ documentation system. 

How to use
----------
#. Download `jal-1.0.zip <http://www.github.com/vimalkumarvelayudhan/sphinx-themes/download/jal-1.0.zip>`_ and uncompress the archive.
#. Modify the ``conf.py`` of an existing Sphinx project or create a new 
   Sphinx project using ``sphinx-quickstart``.
#. Change the ``html_theme`` parameter to ``jal``.
#. Change the ``html_theme_path`` to location of the downloaded theme.
